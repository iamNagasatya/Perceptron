Perceptron Python Package

Welcome to the documentation page for the Perceptron Python package. This package provides a simple implementation of the perceptron algorithm for binary classification tasks.

Installation

To install the Perceptron package, you can use pip:

pip install perceptron

Usage

Here's an example of how to use the Perceptron package:

from perceptron import Perceptron

# Create a perceptron object
p = Perceptron()

# Train the perceptron
p.train(X, y)

# Make predictions
predictions = p.predict(X_test)
